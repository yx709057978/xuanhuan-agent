# 第21集：烟花灿烂的夜晚

※ 青云宗·广场

火光冲天。
地火灵脉的爆发像一条愤怒的火龙，瞬间吞噬了提纯房，并向四周蔓延。
林枭被气浪掀飞，身上的护体灵光剧烈闪烁。
林枭（暴怒）：“疯子！都是疯子！护山大阵给我开！”

趁着混乱，陈十一用缩骨功挣脱了绳索（其实早就解开了，一直在演）。
他一把扛起同样被震飞的沈红药。
沈红药（虚弱）：“……真的炸了？”
陈十一看着漫天火光，咧嘴一笑，满脸黑灰。
陈十一：“我说过，要请你通过看烟花。怎么样，够不够大？”
沈红药看着那毁灭性的美景，眼中第一次有了光彩。
沈红药：“……够大。”

陈十一：“走咯！再不走就要变成烤猪了！”
他背起沈红药，趁着林枭忙着镇压地火，钻进了一条早已挖好的地道——那是他修提纯炉时顺手挖的逃生通道，直通后山禁地之外。

【卡黑】
字幕：毁灭，有时候也是一种新生。

# 第22集：也是一种“自由”

※ 荒野·深夜

两人从地道口爬出来，灰头土脸，像两只刚出土的土拨鼠。
身后，青云宗的方向火光映红了半边天。
沈红药回头看了一眼，眼神复杂。那里曾是她的家，也是她的牢笼。
沈红药：“我们……去哪？”
陈十一拍了拍身上的土，从怀里掏出半个馊馒头，掰了一半递给她。
陈十一：“天大地大，哪都行。只要没有林枭，没有KPI，就是乱葬岗也是天堂。”

沈红药接过馒头，咬了一口，很难吃，但她却觉得这是这辈子吃过最香的东西。
风吹过，带着荒野的血腥味和焦糊味。
陈十一深吸一口气，张开双臂。
陈十一：“闻闻，这就是自由的味道……虽然有点臭。”
两人相视一笑，跌跌撞撞地走向黑暗的荒野深处。

【卡黑】
字幕：自由的代价是流浪，但总比圈养强。

# 第23集：遭遇赵二狗

※ 黑市外围·枯树林

几天后。两人衣衫褴褛，如同乞丐。
沈红药的伤势加重，急需疗伤丹药。陈十一决定带她去附近的黑市碰碰运气。
刚进树林，一个穿着花花绿绿道袍、杀马特发型的青年跳了出来。
手持一把生锈的鬼头刀，摆了个并不帅气的poss。
赵二狗：“此山是我开，此树是我栽！要想从此过，留下……留下买路财！”

陈十一看着这个只有练气九层的“劫匪”，不仅没怕，反而有些亲切。
这才是底层修真界的烟火气啊。
陈十一：“兄弟，你看我们像有钱人吗？我们连裤衩都快破洞了。”
赵二狗打量了一下两人，嫌弃地撇撇嘴。
赵二狗：“也是，比我还穷。算了，把那个女的留下，虽然病恹恹的，但卖给合欢宗还能值两块灵石。”

陈十一的眼神瞬间冷了下来。
他依然在笑，但手已经摸向了腰间的暗袋。
陈十一：“兄弟，做人留一线。你要钱我可以给你，但你要人……那就只能给你这个了。”
赵二狗：“什么？”
陈十一掏出一个竹筒状的东西，对准了赵二狗的脸。

【卡黑】
字幕：永远不要小看一个手艺人手中的“工具”。

# 第24集：辣椒水的滋味

※ 黑市外围·枯树林

赵二狗还没反应过来，陈十一按下了竹筒上的机括。
“滋——！！！”
一道红色的水柱喷射而出，精准地滋在赵二狗脸上。
那不是普通的水，而是陈十一用“烈火椒”提炼的高浓度辣椒水，混合了腐蚀性毒液。

赵二狗：“啊啊啊！我的眼睛！我的脸！这是什么妖术？水系法术吗？”
他捂着脸在地上打滚，鬼头刀都扔了。
陈十一走过去，一脚踩在他胸口，捡起鬼头刀架在他脖子上。
陈十一：“这叫‘黯然销魂水’。滋味如何？”
赵二狗痛哭流涕：“大哥饶命！我上有八十老母，下有……”
陈十一：“闭嘴。打劫。”

十分钟后。
赵二狗只剩下一条裤衩，瑟瑟发抖地蹲在树下。
陈十一喜滋滋地翻看着从他身上搜刮来的储物袋：几块劣质灵石，几瓶低阶丹药，还有一本破破烂烂的《黑市生存指南》。
陈十一扔给沈红药一瓶疗伤丹。
陈十一：“你看，这不就富裕了吗？这叫‘黑吃黑’，修真界第一定律。”
赵二狗哭丧着脸：“大哥，能不能把裤子还我？这天挺冷的。”
陈十一：“不行，裤子也是资源。这布料不错，能改个口罩。”

【卡黑】
字幕：在丛林里，猎人和猎物的身份随时互换。

# 第25集：碰瓷教学现场

※ 黑市·入口处

有了赵二狗的“赞助”，沈红药的伤势暂时稳住了。但这点钱根本不够后续治疗。
陈十一看着熙熙攘攘的黑市，计上心头。
他把赵二狗抓过来（赵二狗被迫成了跟班）。
陈十一：“二狗，想不想把裤子赢回来？”
赵二狗疯狂点头。
陈十一：“那好，我教你一招‘无本万利’的生意——碰瓷。”

赵二狗：“碰……瓷？”
陈十一指着前方一个穿着华丽、一看就是暴发户的胖修士。
陈十一：“看好了。第一步，寻找目标。要找那种有钱、怕事、爱面子的。那个胖子就是绝佳的‘肥羊’。”
陈十一整理了一下衣领，大摇大摆地走了过去。
就在与胖子擦肩而过的瞬间，陈十一突然一个踉跄，以一种极其浮夸但又精准的角度撞在胖子身上。
“噗！”
陈十一喷出一口鲜血（其实是鸡血），倒飞出去三米远，捂着胸口抽搐。
陈十一（凄厉）：“哎哟！我的道基！我的金丹！碎了！全碎了！”

胖子懵了：“我……我没用力啊？你就练气期，哪来的金丹？”
周围的人迅速围了上来。
陈十一一边吐血一边指着胖子：“你赔！这是我祖传的‘隐形金丹’！今天不赔个一千灵石，这事没完！”
赵二狗在旁边看得目瞪口呆。
【独白】（赵二狗）：原来……修仙还能这么修？

【卡黑】
字幕：只要你敢躺下，世界就是你的提款机。
