# 第6集：KPI翻倍的绝望

※ 青云宗·主殿

气氛压抑得像要下雨。
吴算子在祖师爷画像前焦躁地踱步，手里盘着的劣质灵石都被捏出了裂纹。
吴算子（咆哮）：“翻倍？玄天宗是疯了吗？这个月就要翻倍？”

一张漆黑的传讯符悬浮在半空，传出林枭冰冷的声音。
【画外音】（林枭）：“下个月起，青云宗的上供额度提升至两千枚下品灵石。交不够，就拿人头来凑。你们宗门，正好还有三个人。”

传讯符燃尽。
吴算子一屁股坐在太师椅上，瞬间老了十岁。
他看向角落里的沈红药和陈十一。
吴算子（无力）："听到了？翻倍。机器不歇人也不歇，把所有备用炉都开了。红药，你去……再去想想办法。"
沈红药脸色一白，下意识摸了摸手臂，那是抽血留下的针孔位置。
沈红药（麻木）："是，师父。"

陈十一靠在柱子上，嘴里的狗尾草也不香了。
【独白】（陈十一）：两千枚？把这老头卖了也凑不齐啊。这哪是修仙，这是催命。

【卡黑】
字幕：当生存变成数字游戏，人命就成了唯一的筹码。

# 第7集：师姐的秘密

※ 深夜·后山废弃丹房

月光惨白。陈十一鬼鬼祟祟地跟在沈红药身后。
他施展“壁虎游墙功”，无声无息地挂在房梁上。
下方，沈红药卷起袖子，露出一截如藕段般的手臂——上面密密麻麻全是针眼，有些已经溃烂。

△ 沈红药拿出一根粗大的银针，刺入血管。
殷红的鲜血顺着导管流入一个玉瓶。她的脸色以肉眼可见的速度灰败下去，但瓶中的血液却泛着奇异的灵光。
一个黑袍人（黑市药商）从阴影中走出，接过玉瓶，扔下一袋灵石。
黑袍人：“筑基期‘炉鼎’的精血，成色越来越差了。这次只能给八折。”
沈红药（声音沙哑）：“……好。”

房梁上，陈十一瞳孔微缩。
他知道“炉鼎”意味着什么——被抽取本源，直到干枯而死。
黑袍人走后，沈红药瘫坐在地，拿出一颗劣质回血丹吞下，剧烈咳嗽，咳出的全是血沫。
陈十一看着这一幕，握紧了手中的开锁铁丝，又松开。
【独白】（陈十一）：救她？拿什么救？我自己都还欠着一屁股债。

突然，沈红药猛地抬头，看向房梁。
沈红药：“看够了吗？”
陈十一尴尬地跳下来，挠挠头：“那啥，今晚月色不错，我来赏月……顺便问问，这血能卖多少钱？我也想卖点。”
沈红药冷冷看着他，半晌，嘴角扯出一丝嘲讽的弧度。
沈红药：“你是杂役，血里全是杂质，狗都不喝。”

陈十一耸耸肩，走到她身边，递过一块干净的手帕。
陈十一：“擦擦吧，嘴角的血太显眼，不知道的还以为你偷吃了番茄酱。”
沈红药接过手帕，没说话。两人并肩坐在废墟里，月光将影子拉得很长。

【卡黑】
字幕：在这个世界，连卖命都要看成色。

# 第8集：猪草与炼丹术

※ 提纯房·次日

陈十一看着那一堆原石发愁。翻倍的任务量，就算把炉子修出花来也完不成。
沈红药虚弱地靠在墙边，脸色比死人还难看。
陈十一突然一拍大腿。
陈十一：“既然量不够，那就凑数呗！”

吴算子凑过来：“怎么凑？石头变灵石？”
陈十一嘿嘿一笑，指着后山漫山遍野的一种紫色野草——紫幽草，俗称“猪草”，因为含有微弱毒性，连兔子都不吃，只有猪勉强能嚼。
陈十一：“这玩意儿汁液是紫色的，灵石也是紫色的。咱们把它捣碎了，混合一点灵石粉末，再加点面粉……”
吴算子瞪大眼睛：“造假？林枭可是金丹期（伪），他能看不出来？”

陈十一从怀里掏出一本破书——《江湖骗术大全·丹药篇》。
陈十一：“如果是普通的假药，肯定露馅。但如果我们用‘障眼法’呢？我有一种秘制‘包衣’技术，能锁住灵气波动。只要不咬开，看起来比真灵石还真！”
吴算子犹豫了。他是个奸商，但他没胆子骗玄天宗。
沈红药突然开口：“我同意。”
她看着吴算子，眼神决绝：“师父，我的血快流干了。如果不拼一把，大家都要死。”

吴算子咬咬牙，眼中闪过一丝狠厉。
吴算子：“干了！反正横竖是死，不如搏一把！”

【卡黑】
字幕：当正道走不通时，邪路就成了唯一的生路。

# 第9集：影帝级的造假现场

※ 青云宗·主殿

大殿被改造成了临时“造假作坊”。
吴算子负责捣碎猪草，沈红药负责搓丸子，陈十一负责最核心的“包衣”工序。
陈十一运指如飞，将灵石粉末均匀地裹在猪草丸子表面，然后打出一道微弱的灵力符文——这是他偷学的低阶“封灵符”。

【特效】丸子瞬间散发出诱人的紫色光泽，灵气氤氲，竟然还有所谓的“丹纹”（其实是陈十一用指甲划的）。
吴算子拿起一颗，惊叹不已。
吴算子：“妙啊！这卖相，比下品灵石还像中品灵石！陈十一，你这手艺不去造假币真是屈才了。”
陈十一擦擦汗：“过奖过奖，祖传手艺，童叟无欺。”

三人像流水线一样疯狂生产。
沈红药一边搓，一边忍不住想笑。
沈红药：“如果林枭知道他在吃猪草……”
陈十一：“那是他的福气，纯天然无公害，富含膳食纤维。”
气氛竟然有些诡异的温馨。这是青云宗第一次像一个“团队”。

夜深了。
一筐筐“高仿灵石”堆满了大殿。
吴算子看着这些假货，突然有些腿软。
吴算子：“要是……被发现了怎么办？”
陈十一拍拍他的肩膀，眼神坚定。
陈十一：“那就演。人生如戏，全靠演技。师父，您那仙风道骨的范儿，这时候得支棱起来啊！”

【卡黑】
字幕：最完美的谎言，往往连自己都信了。

# 第10集：验货时刻

※ 青云宗·山门外

【付费节点】
乌云压顶，狂风大作。
一艘巨大的白骨飞舟缓缓降落。林枭一身锦袍，身后跟着两名凶神恶煞的执法弟子。
那股高高在上的威压，让吴算子瑟瑟发抖。

林枭落下，目光扫过三人，最后停在陈十一身上，眉头微皱。
林枭：“这就是那个新来的主管？看着像个猴子。”
陈十一立刻换上一副谄媚的笑容，躬身行礼。
陈十一：“上宗大人慧眼如炬，小的就是只猴子，专门给您献桃来了。”

吴算子颤抖着呈上装满“灵石”的储物袋。
吴算子：“两……两千枚，幸不辱命。”
林枭接过储物袋，打开看了一眼，眼中闪过一丝惊讶。
林枭：“成色不错啊，比往年还要好。”
吴算子松了一口气，刚想擦汗。

突然，林枭伸手从袋子里捏出一颗“灵石”。
他放在鼻子下闻了闻，又放在耳边听了听。
全场死寂。
陈十一的心提到了嗓子眼，手心里全是汗，暗扣一枚火雷子。
沈红药的手指也悄悄摸上了袖中的毒针。

林枭嘴角勾起一抹玩味的笑容，缓缓将那颗“灵石”——放入了嘴里。
“咔嚓。”
清脆的咀嚼声。
林枭嚼了两下，眉头舒展，似笑非笑地看着陈十一。
林枭：“这味道……有点像我家猪吃的紫幽草？而且，还有一股面粉味？”

轰——！
吴算子直接瘫倒在地。
陈十一瞳孔地震。
完了。
林枭咽下“灵石”，拍了拍手上的残渣，目光变得冰冷刺骨。
林枭：“你们青云宗，很有创意啊。”

【卡黑】
字幕：有些假，造得太真，反而是一种罪过。
