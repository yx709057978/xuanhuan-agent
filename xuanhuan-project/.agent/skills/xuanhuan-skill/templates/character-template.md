## 【人物配置建议】

根据不同故事类型灵活配置角色：

**废材逆袭+退婚流**：
- 必需：主角、炮灰反派、最终boss
- 建议：未婚妻（打脸对象A）、宗门长老（权威见证）、忠心跟班（成长助力A）

**纯升级+签到流**：
- 必需：主角、炮灰反派、最终boss
- 建议：宗门长老（权威见证）
- 可不要：打脸对象（选项E）、跟班（主角独行更爽）

**重生复仇流**：
- 必需：主角、炮灰反派（前世仇敌）、最终boss（幕后黑手）
- 建议：前队友（打脸对象B）、权威见证
- 可不要：跟班（复仇路上不需要累赘）

**天才对决+战力榜单流**：
- 必需：主角、炮灰反派（各路天才）、最终boss（榜首）
- 建议：裁判（权威见证）
- 可不要：打脸对象（选项E，纯战斗）、跟班

## 【人物设计原则】

- **主角**：境界/身份+处境/背景+性格标签+外貌特征，四要素齐全
- **配角工具人化**：只保留推动情节的关键角色
- **功能>身份**：关注角色的功能而非固定身份
- **灵活>固定**：根据故事需要配置，不是填空题
- **少即是多**：避免为了配角而配角，精简克制
- **字数精简**：60集漫剧节奏快，人物描述要简洁有力
