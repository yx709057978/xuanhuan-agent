# 角色一致性检查清单 (Consistency Checklist)

> 在生成 Sora 视频前和生成后使用此清单确保角色一致性

---

## 生成前检查 (Pre-Generation)

### 1. 角色档案完整性

- [ ] **Core Prompt 已准备**: 有可直接复制的角色描述块
- [ ] **颜色方案已确定**: 主色、副色、点缀色都有具体名称
- [ ] **标志特征已列出**: 至少 3 个必须重复的识别特征
- [ ] **描述足够具体**: 没有使用模糊词汇（"好看的"、"普通的"）

### 2. Prompt 质量检查

- [ ] **特征重复**: 每张 Storyboard 卡片都包含核心特征
- [ ] **颜色一致**: 所有卡片使用相同的颜色词汇
- [ ] **风格一致**: 所有卡片使用相同的风格描述词
- [ ] **体型一致**: 身材描述在所有卡片中保持一致

### 3. 技术设置检查

- [ ] **风格词统一**: 确认所有 prompt 使用相同的风格词
- [ ] **分辨率一致**: 所有片段使用相同的分辨率设置
- [ ] **Production Level 一致**: 保持相同的制作级别

## 常见问题与解决方案

### 问题 1: 角色"变脸"

**症状**: 不同镜头中角色面部特征明显不同

**检查点**:
- [ ] 是否在每张卡片重复了面部描述？
- [ ] 面部描述是否足够具体？
- [ ] 是否使用了一致的风格词？

**解决方案**:
1. 在每张卡片开头添加完整的面部描述
2. 使用更具体的特征词（"sharp jawline" 而非 "nice face"）
3. 考虑使用 I2V 模式固定面部

### 问题 2: 服装颜色漂移

**症状**: 衣服颜色在不同镜头中变化

**检查点**:
- [ ] 是否使用了具体的颜色名？
- [ ] 颜色词是否在所有卡片中一致？
- [ ] 是否有光线描述影响颜色感知？

**解决方案**:
1. 使用具体颜色名（"faded gray" 而非 "dark"）
2. 在所有卡片中使用完全相同的颜色词
3. 考虑光线对颜色的影响，必要时说明

### 问题 3: 配饰消失

**症状**: 角色的配饰在某些镜头中消失

**检查点**:
- [ ] 是否在每张卡片提及配饰？
- [ ] 配饰描述是否足够显眼？
- [ ] 镜头角度是否能看到配饰？

**解决方案**:
1. 在每张卡片中明确提及重要配饰
2. 强调配饰的视觉特征（颜色、材质）
3. 对于特写镜头，确保配饰在画面中

### 问题 4: 体型不稳定

**症状**: 角色忽胖忽瘦

**检查点**:
- [ ] 是否有明确的体型描述？
- [ ] 体型描述是否在所有卡片一致？
- [ ] 服装是否影响体型感知？

**解决方案**:
1. 使用具体的体型词（"lean athletic build"）
2. 在每张卡片重复体型描述
3. 保持服装款式一致

### 问题 5: 风格不统一

**症状**: 不同镜头的艺术风格不一致

**检查点**:
- [ ] 是否在所有卡片使用相同的风格词？
- [ ] Production Level 设置是否一致？
- [ ] 是否混用了不同的风格参考？

**解决方案**:
1. 创建风格描述块，在所有 prompt 中复用
2. 保持 Sora 设置一致
3. 避免在不同镜头使用不同的风格参考

## 快速参考卡

### 必须在每张卡片重复的内容

```
1. 体型描述 (lean/muscular/etc.)
2. 面部标志特征 (scar/birthmark/etc.)
3. 发型发色
4. 服装主色
5. 风格词
```

### Prompt 模板

```
[风格词], [体型] [年龄] [性别], [面部特征], [发型发色], 
wearing [服装描述], [配饰], [当前动作/表情]
```

### 检查口诀

> **"型、脸、发、衣、风"**
> 
> - 型：体型一致
> - 脸：面部特征重复
> - 发：发型发色不变
> - 衣：服装颜色统一
> - 风：风格词相同
