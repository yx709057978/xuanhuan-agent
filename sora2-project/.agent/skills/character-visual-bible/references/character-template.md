## 使用说明

1. **新建角色**: 复制上方模板，填写所有相关字段
2. **快速角色**: 对于配角，使用"快速卡片模板"即可
3. **保持更新**: 随着故事发展，更新状态变体
4. **复用描述**: 每次生成 prompt 时，复制 Core Prompt 描述块
5. **检查一致性**: 生成前使用检查清单确认

## 示例：填写后的快速卡片

```markdown
# 陈十一 视觉卡

**类型**: 主角
**一句话描述**: 瘦削青年，破旧灰衣，乱发遮眼，贼眉鼠眼

## Core Prompt (复制用)
```
A lean young man in his early twenties, gaunt face with sharp 
cheekbones, messy black hair falling over his left eye, wearing 
a faded gray tunic with torn sleeves and patches, dark brown 
leather belt, black cloth pants, worn cloth shoes. Shifty eyes, 
alert posture, moves with quiet agility.
```

## 颜色方案
- 主色: faded gray (服装)
- 副色: dark brown (皮带)
- 点缀: dull bronze (扣子)

## 必须重复的特征
1. 乱发遮左眼 (messy hair over left eye)
2. 破旧灰色布衣 (faded gray tunic)
3. 瘦削身材 (lean gaunt build)

## 常用状态
- 默认: 警觉的小偷姿态
- 狼狈: 衣服更破，有灰尘血迹
- 伪装: 换上偷来的干净衣服
```
