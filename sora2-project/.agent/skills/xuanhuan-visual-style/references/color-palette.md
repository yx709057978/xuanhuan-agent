## 色调调整指南

### 时间段色调偏移

| 时段 | 整体偏移 | 说明 |
|-----|---------|------|
| 深夜 | +蓝 -饱和 | 冷蓝调，低饱和 |
| 黎明 | +粉蓝 | 微弱的暖色开始 |
| 白天 | 基准 | 依然偏灰 |
| 黄昏 | +橙 | 难得的暖色 |
| 傍晚 | +紫蓝 | 过渡到夜晚 |

### 情绪色调偏移

| 情绪 | 整体偏移 | 说明 |
|-----|---------|------|
| 紧张 | +对比 -饱和 | 更极端的明暗 |
| 绝望 | +蓝 -饱和 | 冷色调，几乎黑白 |
| 荒诞 | 略+饱和 | 稍微鲜艳以突出讽刺 |
| 温暖 | +暖 +饱和 | 仅限弃仙谷等场景 |
| 疯狂 | +紫红 +对比 | 不稳定的色彩 |

## Sora 色彩描述模板

### 默认色调

```
"Desaturated color palette dominated by grays and browns,
occasional warm orange from fire, cold blue from moonlight,
high contrast, gritty texture, NOT vibrant or colorful"
```

### 夜景色调

```
"Night scene with cold blue moonlight, deep black shadows,
desaturated colors, only warm highlights from torches,
noir-like contrast, atmospheric haze"
```

### 爆炸场景色调

```
"Explosive scene with bright orange and yellow flames,
black smoke, white shockwave, high contrast between
blinding light and pitch darkness, debris in warm tones"
```

### 弃仙谷色调（例外）

```
"Warmer color palette with golden sunlight, green vegetation,
brown earth tones, still slightly desaturated compared to
normal, fragile warmth in a cold world"
```
