## 组合示例

### 赛博朋克雨夜

```
Atmosphere: Gritty, neon-drenched, melancholic yet vibrant
Visual Style: Neo-noir cyberpunk, high contrast
Color Palette: Deep blacks, electric cyan, hot pink neon, 
              rain-soaked reflections
Materials: Wet asphalt reflecting neon, chrome surfaces, 
          holographic advertisements
Physical Effects: Heavy rain, steam rising from vents, 
                 neon signs flickering, puddles rippling
Mood: Isolated in a crowd, technological alienation, 
      beautiful decay
```

### 废弃神殿黄昏

```
Atmosphere: Sacred yet forgotten, melancholic grandeur
Visual Style: Cinematic, warm golden hour lighting
Color Palette: Amber gold, deep shadow, moss green, 
              weathered stone gray
Materials: Cracked marble, overgrown vines, dust-covered 
          statues, faded frescoes
Physical Effects: Dust motes in golden light beams, 
                 leaves drifting through broken roof,
                 birds nesting in corners
Mood: Reverent silence, time's passage, faded glory
```

### 未来实验室

```
Atmosphere: Clinical, sterile, unsettling precision
Visual Style: Sci-fi minimalist, cool blue tones
Color Palette: Stark white, cold blue, chrome silver, 
              warning red accents
Materials: Smooth white surfaces, glass containers, 
          holographic displays, brushed metal
Physical Effects: Soft hum of machinery, floating 
                 holographic data, sterile air circulation
Mood: Detached, controlled, underlying tension
```
