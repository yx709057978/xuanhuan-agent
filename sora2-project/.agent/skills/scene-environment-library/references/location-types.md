## 使用技巧

### 组合描述公式

```
[建筑风格] + [场景类型] + [状态] + [规模] + [特殊细节]
```

### 示例组合

**现代都市**:
```
Sleek minimalist penthouse apartment, floor-to-ceiling windows 
overlooking neon-lit cyberpunk cityscape, sparse modern furniture, 
cold blue ambient lighting, rain streaking down glass walls.
```

**废弃工业**:
```
Abandoned brutalist warehouse, crumbling concrete walls with rust stains,
broken skylights letting in shafts of dusty light, scattered debris 
and old machinery, puddles on the cracked floor reflecting dim light.
```

**奇幻古堡**:
```
Gothic castle throne room, towering stone columns with carved gargoyles,
faded tapestries on walls, dust motes floating in colored light from 
stained glass windows, ancient stone floor worn smooth by centuries.
```
