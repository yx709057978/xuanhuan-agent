# 节奏控制指南

> 分镜脚本的时间线规划与节奏控制

---

## 基础节奏理论

### 镜头时长与情绪

| 镜头时长 | 情绪效果 | 适用场景 |
|---------|---------|---------|
| 1-2 秒 | 紧张、混乱、快节奏 | 打斗、追逐、惊吓 |
| 3-4 秒 | 正常、自然 | 对话、日常 |
| 5-7 秒 | 沉稳、沉思 | 情绪渲染、悲伤 |
| 8+ 秒 | 史诗、庄重、留白 | 开场、结尾、震撼 |

### Sora 2 时长限制

| 版本 | 最大时长 | 建议单场景时长 |
|-----|---------|--------------|
| 标准版 | 10-15 秒 | 10 秒 |
| Pro 版 | 20-25 秒 | 15-20 秒 |

## 场景类型节奏模板

### 1. 动作/打斗场景

**特点**: 快节奏、多切换、强冲击

```
时间线示例 (15秒):
├─ 0:00-0:02 [全景] 双方对峙
├─ 0:02-0:03 [特写] A 眼神
├─ 0:03-0:04 [特写] B 眼神
├─ 0:04-0:06 [全景] 双方冲向对方
├─ 0:06-0:07 [中景] 第一次交锋
├─ 0:07-0:08 [特写] 武器碰撞火花
├─ 0:08-0:10 [全景] 连续攻防
├─ 0:10-0:12 [中景] 关键一击
├─ 0:12-0:13 [特写] 被击中反应
└─ 0:13-0:15 [全景] 结果揭示
```

**Sora Card 间距**: 1-2 秒，紧密排列

---

### 2. 对话场景

**特点**: 中等节奏、正反打、情绪递进

```
时间线示例 (12秒):
├─ 0:00-0:03 [中景双人] 建立场景
├─ 0:03-0:05 [过肩A] A 说话
├─ 0:05-0:07 [特写B] B 反应
├─ 0:07-0:09 [过肩B] B 回应
├─ 0:09-0:11 [特写A] A 反应
└─ 0:11-0:12 [中景双人] 收尾
```

**Sora Card 间距**: 2-3 秒，适中

---

### 3. 情绪/氛围场景

**特点**: 慢节奏、长镜头、留白多

```
时间线示例 (15秒):
├─ 0:00-0:05 [远景] 环境空镜，缓慢移动
├─ 0:05-0:10 [中景] 角色入画，静止或缓慢动作
└─ 0:10-0:15 [特写] 面部表情，情绪渲染
```

**Sora Card 间距**: 4-5 秒，宽松

---

### 4. 悬疑/恐怖场景

**特点**: 慢-快交替、突然切换

```
时间线示例 (12秒):
├─ 0:00-0:04 [中景] 角色缓慢探索 (慢)
├─ 0:04-0:06 [特写] 发现异常 (慢)
├─ 0:06-0:07 [特写] 惊恐表情 (快)
├─ 0:07-0:08 [全景] 威胁揭示 (快)
└─ 0:08-0:12 [追逐/逃跑] 快节奏 (快)
```

**Sora Card 间距**: 变化，慢段 3-4 秒，快段 1 秒

---

### 5. 揭示/震撼场景

**特点**: 铺垫-爆发、节奏突变

```
时间线示例 (10秒):
├─ 0:00-0:03 [中景] 角色接近/准备
├─ 0:03-0:05 [特写] 期待/紧张表情
├─ 0:05-0:06 [黑屏/闪白] 过渡 (可选)
├─ 0:06-0:08 [全景] 震撼揭示
└─ 0:08-0:10 [特写] 反应
```

**Sora Card 间距**: 铺垫段 2-3 秒，揭示瞬间紧密

## 节奏变化技巧

### 加速 (Accelerating)

用于：紧张升级、高潮逼近

```
镜头时长变化: 4s → 3s → 2s → 1s → 1s
效果: 越来越紧张
```

### 减速 (Decelerating)

用于：高潮后平复、情绪沉淀

```
镜头时长变化: 1s → 2s → 3s → 5s
效果: 逐渐平静
```

### 突变 (Sudden Change)

用于：惊吓、转折、震撼

```
镜头时长变化: 5s → 5s → 1s (突然)
效果: 打破预期，制造冲击
```

### 呼吸感 (Breathing)

用于：自然节奏、避免单调

```
镜头时长变化: 3s → 2s → 4s → 2s → 3s
效果: 有起伏但不极端
```

## 玄幻短剧节奏建议

### 第一幕：生存 (紧张+荒诞)

```
整体节奏: 中等偏快
动作场景: 1-2秒/镜头
对话场景: 2-3秒/镜头
荒诞喜剧: 3-4秒/镜头 (留出反应时间)
```

### 第二幕：挣扎 (压抑+爆发)

```
整体节奏: 慢-快交替
压抑场景: 4-5秒/镜头
爆发场景: 1-2秒/镜头
情绪场景: 5-7秒/镜头
```

### 第三幕：沉沦 (疯狂+毁灭)

```
整体节奏: 逐渐加速到极致
前期: 3-4秒/镜头
中期: 2-3秒/镜头
高潮: 1秒/镜头
结尾: 5-8秒/镜头 (留白)
```

## Sora 2 Storyboard 间距建议

### 卡片间距与效果

| 间距 | 效果 | 适用 |
|-----|------|-----|
| 0-1秒 | 硬切，跳跃感 | 动作、惊吓 |
| 2-3秒 | 自然过渡 | 大多数场景 |
| 4-5秒 | AI 会填充细节 | 需要过渡动画 |
| 6秒+ | AI 大量填充 | 不推荐 |

### 最佳实践

```
推荐间距: 2-4 秒
最大间距: 5 秒
最小间距: 1 秒 (仅用于硬切)
```
