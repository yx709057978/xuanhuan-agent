## 使用说明

1. **拆解时**：每拆解6章，追加一个新批次到"剧情列表"
2. **分集时**：拆解时就直接标注"第X集"
3. **状态管理**：
   - 拆解后默认：状态：未用
   - 创作剧本后：状态：未用 → 状态：已用
4. **剧情编号**：连续编号，从【剧情1】开始
