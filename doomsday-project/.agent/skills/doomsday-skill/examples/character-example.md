# 人物小传

## 主角（必需）

**林念**（女主）

前世是普通房产中介，被男友陈俊PUA三年后在极寒末日第五年被抛弃，冻死在零下80度的街头；今生重生携带五年完整记忆，28岁，气质疏离冷淡，眼神锐利如冰。冷静理智不圣母、腹黑复仇记仇狠、专业能力强。金手指：前世五年记忆+房产结构专业眼光+改造设计能力。复仇目标：让前世所有害过她的人在极寒中付出代价，尤其是渣男陈俊；同时与叶澜结盟，建立女性互助的新世界。前世与叶澜互相伤害，今生成为最强盟友；对陈俊恨之入骨但表面冷漠；对前世恩人有恩必报。

**叶澜**（女主/闺蜜）

前世是富二代会计师，因恋爱脑被陈俊利用，在他PUA下背叛林念、抢夺物资，最终同样被抛弃冻死；今生重生携带五年完整记忆，26岁，气质优雅但眼神犀利，长发及腰。温柔外表狠辣内核、擅长算计与布局、对爱情彻底觉醒。金手指：前世五年记忆+三百万遗产+资金运作能力+供应链管理经验。复仇目标：与林念合作扳倒陈俊，报前世之仇；用钱和智慧为女性开辟新路。前世害过林念但今生坦诚道歉、结成死党；对陈俊从爱到恨；负责基地的资金和物资管理。

## 炮灰反派（必需）

**说明**：各阶段被碾压的龙套，每批5-10集出现一批，打完就死，不重复利用。

**李建国**（第11-15集）

林念的前邻居，五十多岁，秃顶啤酒肚。贪婪自私、见风使舵。前世末日初期抢过林念的物资，还反咬一口说她偷东西。今生极寒降临后带着全家上门求救，被林念冷眼拒绝。在暴风雪中冻死在别墅门外，死前还在诅咒林念。

**王芳**（第11-15集）

林念的前同事，三十岁出头，妆容精致。表面客套内心刻薄。前世在公司造谣林念勾引客户，导致她失业。今生极寒降临后找到林念，想用前同事身份套近乎。被林念当场揭穿前世所为，狼狈离开后在抢劫中被杀。

**赵老板**（第16-20集）

小区物业经理，四十多岁，一脸横肉。势利狗眼看人低。前世因林念交不起物业费而断水断电，间接害死她。今生带着物业保安想强行征用林念的别墅。被林念和叶澜的防御系统震慑，灰溜溜离开后在极寒中饿死。

**刘队长**（第21-25集）

官方避难所派来的军官，三十五岁左右，国字脸。官僚主义、仗势欺人。奉命来"征用"林念叶澜的物资和设施，态度强硬。被两人展示的武力和先知能力震撼，回去后被上级问责，在避难所内部斗争中被清洗。

**孙大虎**（第26-30集）

流民团伙头目，壮汉，满脸胡茬。凶残暴力、欺软怕硬。带着手下袭击林念叶澜的基地，想抢劫物资。被基地的防御系统击退，在逃跑途中遇到冰封风暴，全团覆灭。

## 前世仇人（必需 - 打脸复仇的对象）

**说明**：这是复仇打脸的核心对象。

### 渣男（核心仇人）

**陈俊**

29岁，外貌斯文帅气、戴金丝眼镜，实则冷血自私的PUA高手。表面深情体贴、善于伪装，实则把女人当工具，擅长同时操控多个女性为他服务。前世利用林念的感情骗取她的信任和物资，又用同样手法拉拢富二代叶澜；在末日中让两人互相争斗、帮他囤货；等物资到手后，他抛弃二人，搂着新猎物躲进安全屋，眼睁睁看着林念和叶澜冻死。今生末日前再次出现，用前世那套深情话术想复合，被林念冷淡拒绝；又去找叶澜，同样碰壁。极寒降临后，他失去所有物资，狼狈不堪地来求救，被林念当面揭穿他前世的罪行："前世你怎么对我的，记得吗？"被拒后震惊恐惧，试图强闯被驱离。最终在零下70度的暴风雪中，像前世抛弃她们一样被抛弃，冻死在林念别墅外，死前想起前世所有罪恶，充满恐惧和后悔。

[注：本作品中，林念和叶澜前世互为仇人，但今生已和解结盟，不再是打脸对象。]

## 前世恩人（可选 - 展现主角恩怨分明）

**说明**：展现主角不是纯恶，有恩必报。

**周奶奶**

七十多岁，独居老人，头发花白但精神矍铄。善良坚韧、不求回报。前世末日初期，林念被陈俊抛弃后流落街头，是周奶奶分给她仅有的半碗热粥，让她多活了两天。今生极寒降临后，林念主动找到她，接她进入安全屋，悉心照顾。周奶奶避免了前世冻死的命运。

**小宇**

12岁孤儿，瘦小但机灵，大眼睛透着聪慧。懂事早熟、学习能力强。前世末日中曾救过叶澜一次，把自己捡到的食物分给她。今生极寒降临后，叶澜找到他，收养并保护他。小宇在基地中帮忙做杂事，对两位姐姐感恩戴德。

## 队友/家人（可选 - 保护对象）

**说明**：协助主角建立基地的可靠队友。

### 忠诚队友（适合基建流、团队流）

**高工**

42岁，前建筑工程师，沉稳寡言，戴厚重眼镜。专业能力强、忠诚可靠、不问私事。前世是避难所的技术骨干，因太过正直被权贵排挤。今生被林念叶澜提前拉拢，负责基地的加固和防御系统建设，对两人的先知能力从不多问，默默贡献专业技能。

**李医生**

35岁，外科医生，清瘦干练，眼神温和。医术精湛、心思细腻、重情重义。前世在避难所救治过无数人，最后因物资匮乏累死。今生被林念叶澜救下，负责基地的医疗，对救命之恩铭记于心，誓死效忠。

**老兵**

50岁，退伍军人，身材魁梧，脸上有疤。战斗经验丰富、纪律严明、疾恶如仇。前世组织过幸存者抵抗变异生物，战死沙场。今生被两人招募，负责基地的安保和战斗训练，对两人的指挥佩服得五体投地。

## 最终对手（必需）

**张教授**

55岁，避难所首席科学顾问，斯文儒雅、头发花白，实则野心勃勃、冷酷无情。气候学家出身，掌握气候武器核心技术，表面为人类谋福利，实则是极寒末日的幕后黑手。前世秘密操控气候武器，制造极寒末日以实施人口清洗计划，在混乱中建立独裁政权。林念和叶澜前世发现真相后试图揭露，被他设局杀害。今生两人重生后，凭借前世记忆逐步接近真相。张教授察觉异常，布局陷害，与两人展开智斗。第40集被揭露罪行后，临死前启动气候武器自毁程序，企图让全人类陪葬。实力体现：掌控避难所高层、操控气候武器、精通政治手腕。最终对决：林念叶澜揭露其罪行，他启动自毁程序后被击毙，两人必须前往北极手动关闭武器。

## 【人物配置说明】

本作品为**双女主重生+囤货复仇+基建流**，人物配置如下：

**核心角色**：
- 双女主：林念+叶澜（前世仇敌，今生盟友）
- 核心仇人：陈俊（渣男，PUA双女主的罪魁祸首）
- 终极Boss：张教授（气候武器操控者，前世害死双女主）

**功能角色**：
- 炮灰反派5批（各阶段打脸对象）

- 林念↔叶澜：前世互害，今生结盟，姐妹情深
