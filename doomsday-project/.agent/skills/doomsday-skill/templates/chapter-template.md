**【视觉描述符号】**

**基础符号**（通用）
- **※** 标注场景/环境
- **△** 标注动作/变化
- **【特效】** 标注特效（空间光芒、天灾效果等）
- **【文字】** 标注画面文字（天灾阶段、物资数值等）
- **【音效】** 标注音效（警报声、风雪声、BGM等）
- **【独白】** 内心独白（简短，不超过10字）
- **【卡黑】** 悬念结尾（每集必须）

**末世特色符号**
- **【空间面板】** 展示囤货物资和空间功能
- **【闪回】** 前世今生对比场景
- **【对比】** 主角vs外界的画面对比
- **【蒙太奇】** 快速展示囤货/天灾进程
- **【新闻】** 新闻播报（天灾预警、灾情通报）

**【使用示例】**

```
※ 末世第三年，极寒暴雪中

△ 苏念跪在防盗门外，浑身颤抖
【特效】暴雪呼啸，气温-40度

【闪回】前世：舅舅关上门："粮食不够了，你自己想办法！"

---

※ 熟悉的卧室，阳光明媚

△ 苏念猛然睁眼，大口喘息

【独白】这一次，我要让你们付出代价。

```
