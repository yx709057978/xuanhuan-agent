## 【人物配置建议】

根据不同故事类型灵活配置角色：

**囤货复仇+背叛流**：
- 必需：主角、炮灰反派、前世仇人（渣男未婚夫+白莲花闺蜜）、终极Boss
- 可选：前世恩人、父母家人
- 可不要：队友（独行更爽）

**纯囤货+天灾流**：
- 必需：主角、炮灰反派、终极Boss（变异兽王）
- 可选：父母家人
- 可不要：前世仇人（选项E）、队友（专注囤货）

**基建+势力流**：
- 必需：主角、炮灰反派、前世仇人、终极Boss
- 可选：忠诚队友、前世大佬
- 可不要：家人（格局太小）

**先知+抢机缘流**：
- 必需：主角、炮灰反派、终极Boss
- 可选：前世大佬、忠诚队友
- 可不要：前世仇人（专注抢机缘）

**咸鱼躺平+独行流**：
- 必需：主角、炮灰反派、终极Boss
- 可选：无
- 可不要：前世仇人、队友、家人（主角只想躺平）

## 【人物设计原则】

- **主角**：前世身份+死亡原因+性格标签+外貌特征，四要素齐全
- **配角工具人化**：只保留推动情节的关键角色
- **功能>身份**：关注角色的功能而非固定身份
- **灵活>固定**：根据故事需要配置，不是填空题
- **少即是多**：避免为了配角而配角，精简克制
- **字数精简**：60集漫剧节奏快，人物描述要简洁有力
- **恩怨分明**：主角不圣母，有仇报仇，有恩报恩
- **仇人必死**：前世仇人必须付出代价，下场要凄惨
